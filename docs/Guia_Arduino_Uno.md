# Arduino Uno - Guia Técnico S122
`#Hardware` `#Arduino` `#Fundamentos` `#Eletrônica`

## 1. Visão Geral (O Básico)
**Para que serve:** Microcontrolador de arquitetura simples, ideal para aprendizado de lógica de programação e eletrônica digital básica sem a complexidade inicial de redes.
**Contexto M3F:** [X] Percepção (Edge) | [ ] Transporte (Fog) | [ ] Aplicação (Cloud)

## 2. Fluxo de Dados (I/O)
- **Entrada (Input):** Sensores analógicos (LDR, Potenciômetro) e digitais (Botões, Presença).
- **Saída (Output):** Atuadores (LEDs, Relés, Buzzers), Displays LCD, Comunicação Serial.

## 3. Instalação e Configuração

### 🪟 Windows / 🐧 Linux
1.  **IDE Oficial:** [Arduino IDE Download](https://www.arduino.cc/en/software)
2.  **Configuração:** 
    - Conectar o cabo USB.
    - No menu `Tools` > `Board`, selecionar **Arduino Uno**.
    - No menu `Tools` > `Port`, selecionar a porta COM ativa.

### 🌐 Simulação (Wokwi)
O Arduino Uno é a placa padrão de entrada no Wokwi. Não requer instalação de drivers para simulação.

## 4. Integração (Ecossistema S122)
- **Conecta com:** Wokwi para prototipagem rápida.
- **Diferença para ESP32:** O Arduino Uno não possui Wi-Fi ou Bluetooth nativo, sendo usado na disciplina para fundamentos de eletrônica e sensores simples.
